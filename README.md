# Descriptive-Analytics-and-Data-Preprocessing-on-Sales-Discounts-Dataset

## 📌 Task Overview

This Task focuses on performing **Descriptive Analytics** and **Data Preprocessing** on a Sales & Discounts dataset. The main objective is to analyze the dataset using statistical measures, visualize data distributions, identify outliers, and derive meaningful insights that can support further exploratory data analysis and machine learning tasks.

---

## 🎯 Objectives

* Perform descriptive statistical analysis on numerical columns.
* Visualize the distribution of numerical variables.
* Identify outliers and unusual patterns in the dataset.
* Analyze categorical variables and their frequency distribution.
* Preprocess the dataset for future analytical and predictive modeling tasks.

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Description

The dataset contains information related to sales transactions and discount details. It includes both numerical and categorical attributes that are analyzed to understand data patterns and business insights.

---

## 📊 Descriptive Analytics

The following statistical measures were computed for all numerical columns:

* Mean
* Median
* Mode
* Standard Deviation

These measures help in understanding:

* Central tendency of the data.
* Data variability and spread.
* Presence of skewness and extreme values.

---

## 📈 Data Visualization

### 1. Histograms

Histograms were plotted for each numerical feature to analyze:

* Data distribution
* Skewness
* Frequency patterns
* Presence of outliers

### 2. Box Plots

Boxplots were created to:

* Identify outliers
* Analyze the Interquartile Range (IQR)
* Understand data spread and variability

### 3. Bar Charts for Categorical Variables

Bar charts were generated for categorical columns to:

* Visualize category frequencies
* Compare category distributions
* Derive insights from categorical data

---

## 🔍 Key Findings

* Several numerical variables exhibited skewed distributions.
* Outliers were identified in multiple numerical columns using boxplots.
* Categorical features showed varying frequency distributions across categories.
* Statistical analysis provided insights into the central tendency and variability of the dataset.

## 🚀 Project Workflow

1. Load the dataset.
2. Identify numerical and categorical columns.
3. Perform descriptive statistical analysis.
4. Visualize numerical features using histograms and boxplots.
5. Analyze categorical features using bar charts.
6. Interpret findings and summarize insights.
